# TaxSentry 1.1.4 Draft - Model Picker UI

## What changed

- Provider selection now uses a clearer cockpit-style layout with badges for `local`, `oauth`, and `flex`.
- Model selection now shows color-coded labels for `recent`, `recommended`, and `custom`.
- Long model lists can be searched with `s` before choosing a model.
- Manual model entry remains available through `c`.

## User impact

- Faster onboarding when the provider exposes many models.
- Less ambiguity when switching between recently used and suggested models.
- Better visual consistency between provider selection and model selection.

## Notes

- This note covers the UI improvements added after the `1.1.3` release.
- The implementation keeps the old fallback path so onboarding still works if `/v1/models` is unavailable.
